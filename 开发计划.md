## 方案一：存储文档 ID 到文件

### 目标
防止 JSON 元数据丢失后无法恢复文档列表的问题。

### 数据结构

在每个 KB collection 目录下新增 `doc_ids.json`：

```json
{
  "version": 1,
  "documents": [
    {
      "title": "民法典第一编",
      "sourcePath": "/Users/djx/Documents/民法典.pdf",
      "userSelectedType": "law",
      "importedAt": "2026-04-17T18:28:00Z",
      "ids": ["uuid-chunk-1", "uuid-chunk-2", "uuid-chunk-3"]
    }
  ]
}
```

## 实施步骤

### Step 1: 修改 `Resources/scripts/import.js`

在导入完成后，将文档 ID 列表写入 `doc_ids.json`：

```javascript
// 导入结束后
const docIdsPath = path.join(kbPath, 'doc_ids.json');

// 读取现有 doc_ids.json（如果存在）
let existingDocIds = { version: 1, documents: [] };
if (fs.existsSync(docIdsPath)) {
    existingDocIds = JSON.parse(fs.readFileSync(docIdsPath, 'utf-8'));
}

// 追加新文档
for (const doc of parsedDocs) {
    const chunkIds = importResults
        .filter(r => r.title === doc.title && r.status === 'success')
        .flatMap(r => r.ids || []);

    existingDocIds.documents.push({
        title: doc.title,
        sourcePath: doc.sourcePath,
        userSelectedType: doc.userSelectedType,
        importedAt: new Date().toISOString(),
        ids: chunkIds
    });
}

fs.writeFileSync(docIdsPath, JSON.stringify(existingDocIds, null, 2));
```

### Step 2: 新增 Swift 数据结构

在 `Sources/Models/` 下新增文件 `DocIdsMetadata.swift`：

```swift
import Foundation

struct DocIdsFile: Codable {
    let version: Int
    var documents: [DocIdRecord]
}

struct DocIdRecord: Codable {
    let title: String
    let sourcePath: String
    let userSelectedType: String
    let importedAt: String
    let ids: [String]
}
```

### Step 3: 修改 `Sources/Models/KnowledgeBase.swift`

**A. 新增恢复方法：**

```swift
static func recoverFromZvec(kbPath: String) -> KnowledgeBase? {
    let docIdsPath = URL(fileURLWithPath: kbPath).appendingPathComponent("doc_ids.json")
    guard let data = try? Data(contentsOf: docIdsPath),
          let docIds = try? JSONDecoder().decode(DocIdsFile.self, from: data) else {
        return nil
    }

    // 从 docIds 重建 DocumentRecord
    let records = docIds.documents.map { doc -> DocumentRecord in
        DocumentRecord(
            title: doc.title,
            type: DocumentType(rawValue: doc.userSelectedType) ?? .misc,
            pageCount: nil,
            chunkCount: doc.ids.count,
            sourcePath: doc.sourcePath,
            fileSize: 0  // 无法恢复
        )
    }

}
```

## 流程图

```
【导入流程】
解析文档 → 分块 → 写入 Zvec → 保存 doc_ids.json
                                    ↓
                          追加到 doc_ids.json

【启动流程】
load() → JSON 正常 → 返回
         ↓
    JSON 为空
         ↓
    检查 doc_ids.json 是否存在
         ↓ 存在
    recoverFromZvec() → 从 doc_ids.json 读取 IDs → fetchSync(ids) → 重建 DocumentRecord
         ↓ 不存在
    返回空 store（真正丢失）
```
